This folder is for prediction of ORR and OER activity for 2V structure and 1V structure:
paper published in iScience: 10.1016/j.isci.2021.102398
and Journal of Materials Chemistry A: DOI: 10.1039/D0TA06207C

Data folder: original data and cleaned data
Models folder: trained model
Results folder: predicted results

1data-clean.ipynb: clean data
4SA-CNN-1V-ORR.ipynb: choose features for 1V structure ORR, build model and train model
4SA-CNN-2V-ORR.ipynb: choose features for 2V structure ORR, build model and train model
5SA-CNN-1V-OER.ipynb: choose features for 1V structure OER, build model and train model
5SA-CNN-2V-OER.ipynb: choose features for 2V structure OER, build model and train model
